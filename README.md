# Strattopay Backend

Strattopay is a secure, scalable remittance platform that allows users to send money globally with live currency conversion and gamified reward features.

This repository contains the backend API built with **Node.js**, **Express**, **PostgreSQL**, and **Sequelize**, using **JWT** for authentication and **Fixer.io** for real-time currency rates.

---

 **Features**

- User registration & login (JWT auth)
- Recipient management
- Money transfers with live USD to NGN/GHS conversion
- Transaction logging
- Admin access to all transactions
- Modular structure ready for deployment

---

**Tech Stack**

- **Backend**: Node.js, Express
- **Database**: PostgreSQL with Sequelize ORM
- **Auth**: JWT & bcrypt
- **API**: Fixer.io (currency exchange rates)
- **Deployment Ready**: Docker/VPS/Render support
